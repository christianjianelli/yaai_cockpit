# Contribute to ABAP AI tools

This project aims to bring AI capabilities into ABAP, and contributions from the community are highly welcome. Everyone is welcome to contribute.

---

## Ways to Contribute

You can contribute in several ways:

* Reporting and/or fixing bugs
* Suggesting new features
* Improving examples or documentation
* Adding new LLM providers or integrations
* Enhancing core functionalities

---

## Before You Start

* For larger changes, please open an issue first to discuss your idea
* Keep contributions small and focused when possible

---

## Development Setup

### Requirements

* SAP system with ABAP ≥ 7.52
* abapGit installed
* [ABAP AI tools](https://github.com/christianjianelli/yaai) installed

### Installation

* [ABAP AI tools Cockpit Installation](./docs/installation.md)

---

## Coding Guidelines

* Follow the existing coding style in the repository
* Use clear and consistent naming
* Keep changes minimal and focused
* Avoid hardcoding provider-specific logic into shared components

---

## Adding LLM Providers

When adding a new provider:

* Follow the structure of existing providers
* Keep provider-specific logic isolated
* Ensure compatibility with existing abstractions

---

## Testing

* Make sure your changes do not break existing functionality

---

## Discussions

If you're unsure about something, open an issue. Questions and ideas are welcome.
